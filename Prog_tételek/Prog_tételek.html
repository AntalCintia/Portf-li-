Mester-es feladatok

A mester-es feladatokban megkaptuk a leírásban a bemenetet és a kimenetet is. Bemenetnek az elején legtöbbször pár paramétert adtak meg, aztán egy tömböt.

3000-es feladatok

Egy maximumkeresés függvény a 3000A feladatból

        
            static int Legnagyobb(List lista)
                {
                    int best = lista[0];
                    for (int i = 1; i < lista.Count; i++)
                    {
                        if (best < lista[i])
                        {
                            best = lista[i];
                        }
                    }
                    return best;
                }

Év végére megkaptuk a 3000-es feladatok hosszú sorát. Az elején leginkább 10 kérdésből álltak a feladatok, de később már olyan is volt, hogy 40 vagy 90 kérdés volt a feladatban. Persze nem mindegyikhez kellett egy külön függvényt írnunk. Volt, hogy egy függvényt 5-10 kérdéshez is fel lehetett használni.

Pszeudo-kód

Pszeudo-kóddal leginkább papíron szoktunk foglalkoztunk. Itt nem feltétlenül az a lényeg, hogy pontosan ugyanazokat a szavakat használd a kódodban, amiket tanultál, hanem hogy érthetően leírd, hogy mit szeretnél programozni.

Papíros dolgozatok

Pszeudo-kódban papíron írtunk dolgozatokat többnyire a 3000-es feladatok alapján.
